# MS SQL Server Management Studio

## Keyboard Shortcuts

* Alt + Shift + Arrow Keys + whatever = multi-line typing

* F5 = Execute

* ctrl+L = Display Estimated Execute Plan
* ctrl+M = Include Actual Execution Plan
* ctrl+R = Window > Show/Hide Results Pane
